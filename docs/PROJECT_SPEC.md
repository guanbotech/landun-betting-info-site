# 蓝盾博彩情报站项目规范

## 站点定位

蓝盾博彩情报站是面向中文用户的博彩资讯、平台资料库、平台排行榜与风险提醒站。站点内容用于资料整理、规则说明、入口核对和风险教育，不提供投注服务，不构成法律、财务或投资建议。

覆盖栏目：

- 电竞博彩
- 体育博彩
- 德州扑克
- 在线游戏
- 平台评测
- 平台排行榜
- 平台资料库
- 风险提醒
- 博彩指南
- 关于我们
- 免责声明

## 内容原则

- 所有正文内容必须能在页面源码中被搜索引擎读取。
- 文章页需要包含标题、描述、标签、发布时间、更新时间、作者、摘要、正文、FAQ、相关文章和免责声明。
- 分类页需要包含栏目说明、重点内容、最新列表、栏目差异化模块、相关平台资料和分页。
- 平台资料页需要包含平台名称、Logo 或字母占位、平台类型、支持内容、风险等级、资料完整度、规则公开度、更新时间、基础资料表格、入口核对说明、风险提醒、FAQ、同类平台推荐和免责声明。
- 排行榜仅基于资料完整度、规则透明度、内容分类和更新时间整理，仅用于资料索引，不构成平台背书。

## 禁用表达

不得使用以下诱导或夸大表达：

- 立即投注
- 立即注册
- 领取奖金
- 马上下注
- Play Now
- 稳赚
- 必胜
- 最安全
- 官方认证
- 百分百可靠

按钮文案优先使用：

- 查看资料
- 阅读评测
- 风险提醒
- 核对入口
- 查看规则
- 阅读全文

## UI 规范

- 视觉基调为深蓝、灰白、橙色风险提醒。
- 呈现金融资讯感、风险情报站感、资料库感。
- 不使用真人、美女、现金、筹码堆、大红大黄赌场风、霓虹灯风格或诱导注册风格。
- 文章封面使用栏目差异化抽象视觉，必须包含栏目名和简短图形元素。
- 卡片、按钮、表格和风险提示需要保持正式、克制、可长期运营的站点质感。

## 移动端规范

- 页面不得横向溢出。
- `body`、`main` 和主要容器在移动端必须正常占满视口宽度。
- 正文左右 padding 不小于 16px。
- 导航在移动端使用折叠菜单。
- 文章卡片、平台卡片、排行榜卡片移动端单列显示。
- 右侧侧边栏移动端放到正文下方。
- H1 标题不得被截断。
- 图片和封面不得撑破容器。
- 表格允许横向滚动。
- Footer 移动端单列显示。

## SEO 规范

- `sitemap.xml` 必须正常生成。
- `robots.txt` 必须允许重要页面抓取。
- 重要页面不得设置 `noindex`。
- 分类页、文章页、平台资料页和排行榜页必须可索引。
- 页面需要 canonical。
- 页面源码需要包含正文，不得把正文做成图片。
- 需要保留面包屑、站内内链、相关文章、FAQ 和免责声明。
- 不得隐藏文字、关键词堆砌或恶意跳转。
