# 第二轮上线前修复

## 背景

当前站点框架方向已确认，需要在上线前完成第二轮 UI、移动端、正式文案和页面真实感优化，避免呈现为模板演示页或开发占位页。

## 修复范围

- 首页 Hero 改为正式站点文案。
- 去除开发说明、模板说明、复刻说明等上线不应出现的文案。
- 优化文章封面占位图，改为栏目差异化抽象视觉。
- 二级栏目页增加栏目专属模块。
- 优化文章详情页阅读宽度、TOC、FAQ、风险提醒、免责声明和相关文章顺序。
- 补强平台资料页字段、基础资料表格、入口核对、同类平台推荐和免责声明。
- 补强排行榜卡片字段和榜单说明。
- Footer 补强五列结构和底部合规说明。
- 新增中文自定义 404 页面。
- 修复移动端横向溢出和内容被压窄问题。
- 保留并检查 sitemap、robots、canonical、正文源码可见性、FAQ、免责声明和可索引状态。

## 验收标准

- `npm run check` 通过。
- 首页、分类页、文章页、平台资料页、排行榜页和 404 页面视觉正常。
- 移动端首页、文章页、平台资料页、排行榜页可读且无横向溢出。
- 禁用文案扫描无命中：立即投注、立即注册、领取奖金、Play Now、稳赚、必胜、最安全、官方认证、noindex。
- `/category/risk-warning` 正常可访问并进入 sitemap。
- 重要页面源码包含正文内容。
