# 关键词资料源与内容计划

本文档用于后续扩展站内原创内容。原则：只整理公开资料，不搬运原文，不把外部来源改成本站来源。文章发布时使用本站自己的中文表达，并在底部标注“根据公开资料整理”或保留参考链接。

## 内容使用原则

- 不复制外站文章正文。
- 不伪造来源，不写“本站独家来源”。
- 外部资料只作为事实核对、规则框架和术语参考。
- 文章正文使用本站原创表达。
- 平台内容只写资料索引和风险核对，不写平台背书。
- 博彩相关内容必须保留年龄、地区法规、财务风险和责任博彩提醒。

## 发布模板

> 本文由蓝盾博彩情报站编辑根据公开资料整理，仅用于资料索引和风险提醒，不构成投注建议或平台背书。博彩存在财务损失和成瘾风险，请先确认所在地法律、年龄限制和个人预算。
