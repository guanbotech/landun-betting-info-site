# 网站图片生成清单

当前站点所有图片建议统一使用 `.webp`，质量 82-90。除首页首屏大背景外，其余封面统一按 1200 x 675 生成，方便在首页、栏目页、文章列表、详情页和移动端复用。

## 通用风格

- 中文博彩资讯站、平台资料库、风险提醒站风格。
- 画面要像正式资讯媒体配图，不要像 Demo 占位图。
- 可以使用轻写实、编辑插画、高级合成图、体育赛事现场感、数据面板。
- 不要出现赌场美女、现金、筹码堆、老虎机、霓虹灯广告风。
- 不要在图里写大段文字。最多只放短标题、栏目名、比分板、数据面板。
- 画面左下或中左区域尽量留一点暗色安全区，因为网站会叠加栏目标签。
- 色彩以深蓝、蓝灰、白色、少量橙色风险提示为主，不要过度花哨。

## 实际显示尺寸参考

- 首页首屏背景：全宽横幅，桌面约 2400 x 1000 比例显示，移动端会裁切，重要主体放右侧中部。
- 首页重点文章大图：接近 16:6.8，图片会被横向裁切，主体不要贴边。
- 普通栏目卡片：16:9。
- 首页频道入口卡：约 16:8.5。
- 专题入口卡：约 16:8.8。
- 平台卡片封面：16:9。
- 平台详情页顶部封面：约 16:7，会裁切上下边缘。
- 博彩资讯文章缩略图：桌面约 160 x 120，移动端约 100% x 144，图片要在小尺寸下也能看出主题。
